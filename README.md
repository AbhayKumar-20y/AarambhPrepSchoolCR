# 📘 Class Report System
A simple web-based **Class Report Generator** built using HTML, CSS, and JavaScript.
This tool helps teachers easily create daily class reports and download them as images.
---
#Features
* 📅 Auto Date with Day
* 🏫 Select Class & Facilitator
* 👨‍🎓 Enter Total, Present & Auto Absent Calculation
* 🧮 Absent Count Validation (Smart Check)
* 📚 Period-wise Entry (1st, 2nd, 3rd…)
* ✍️ Class Work & Home Work Input
* 📌 Absent / HW Not Done / Extra Notes Section
* 🎨 Clean & Professional UI
* 📥 Download Report as Image
---
## 🖥️ Tech Used
* HTML
* CSS
* JavaScript
* html2canvas (for image download)
---

#How to Use
1. Open the website
2. Select Class & Facilitator
3. Enter student details
4. Fill period-wise data
5. Click **Generate**
6. Click **Download Image**
